#大學c++專題，學術與學習用途

#Q-learning:爬行機器人

## **專案簡介**
以舵機作為爬行機器人的動力源，運用 Q-learning 強化學習方法，結合 Arduino 控制平台與超聲波測距模組，使機器人能夠自行學習尋找最適合的移動方式。

## **必備條件**
Arduino、超聲波測距模組、舵機。

## **安裝與執行**(Q表僅供參考)
不建議直接使用，應根據機器人主體，調整Q表及學習參數。
